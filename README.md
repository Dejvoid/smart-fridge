# smart-fridge

## Description
Smart fridge project running on ESP32-CAM and Linux server (Raspberry PI).

### Parts
- ESP32-CAM board
- ILI9488 LCD
- (AHT10 thermometer)
- Raspberry PI or other Linux machine
